# Food-Insecurity-Migration-Analysis
A study of the correlation between food insecurity and migration. Food in(security) represents an involuntary driver of human displacement, causing local environmental and economic crises into larger-scale humanitarian emergencies. Understanding this connection is crucial for developing policy, targeting aid, and managing countries' stability.
Migration & Food Insecurity Analysis

## Overview

This project investigates whether food insecurity is associated with net migration patterns at both global and country levels.

Using FAO undernourishment data and World Bank migration data (2000–2024), the analysis finds no meaningful global linear relationship, but significant variation across countries.

---

## Research Question

Is food insecurity associated with net migration patterns at a global and country level?

---

---

## Key Findings

* Global correlation: **-0.026** → no meaningful relationship
* Country-level variation: **-0.77 to +0.84**
* Migration dynamics are **context-dependent**, not universal
